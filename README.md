## рҹҡҖ Quick Start

### Prerequisites

```bash
# You'll need these retro tools:
- Docker & Docker Compose рҹҗі
- Node.js 18+ & npm рҹ“Ұ
- Git (obviously) рҹҢҝ
- A sense of nostalgia рҹ’ҫ
```

### рҹ”§ Backend Setup (FastAPI + PostgreSQL)

```bash
# Navigate to backend
cd SeeThrough/seethrough-backend

# Fire up the containers (like booting Windows 95... but faster)
docker-compose up -d

# Wait for the magic... вңЁ
# Backend will be running at: http://localhost:8000
```

**Note:** PostgreSQL runs on port `5433` to avoid conflicts. Retro systems had port conflicts too! рҹ”Ң

### рҹҺЁ Frontend Setup (React + TypeScript + Vite)

```bash
# Navigate to frontend
cd frontendbU

# Install dependencies (like inserting floppy disks... but automated)
npm install

# Start the dev server
npm run dev

# Frontend will be running at: http://localhost:5173
```

## рҹҺ® How to Use SeeThrough

### Method 1: Upload Checkout Screenshot рҹ“ё
1. Take a screenshot of a suspicious checkout page
2. Click **"рҹ“Ғ Upload"** button
3. Select your image file (PNG, JPG, etc.)
4. Watch as SeeThrough analyzes it with OCR
5. View detected dark patterns and risk score

### Method 2: Upload HTML Snapshot рҹ“„
1. Save the checkout page HTML (Right-click вҶ’ Save As)
2. Click **"рҹ“Ғ Upload"** button
3. Select the `.html` file
4. SeeThrough parses it for hidden patterns
5. Get instant analysis results

### Method 3: Load Demo Data рҹҺ¬
1. Click **"рҹ’ҫ Load Demo"** button
2. See SeeThrough in action with sample data
3. Explore features without uploading anything
